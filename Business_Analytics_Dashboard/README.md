# 📊 Business Analytics Dashboard

## 📌 Overview

The **Business Analytics Dashboard** is an Excel-based analytics project developed to transform raw business data into meaningful insights through interactive dashboards and visualizations.

This project focuses on analyzing sales, profit, customer segments, product categories, and regional performance using Microsoft Excel's advanced analytical features.

---

## 🎯 Objectives

- Clean and organize raw business data.
- Calculate Total Sales and Profit.
- Analyze monthly business performance.
- Analyze region-wise sales.
- Build an interactive dashboard using Combo Box controls.
- Create meaningful KPIs and charts for business decision-making.
