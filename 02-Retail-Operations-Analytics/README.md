# 🏪 Retail Operations Analytics

## 📌 Overview

The **Retail Operations Analytics** project is an Excel-based business intelligence solution designed to analyze retail operations through **two interactive dashboards**.

The project focuses on monitoring operational efficiency, sales performance, inventory insights, profitability, and category analysis using Advanced Microsoft Excel.

---

## 📊 Dashboard Structure

### 📈 Dashboard 1 – Sales Performance Dashboard

**Focus Areas**

- Sales Performance
- Monthly Sales Trends
- Regional Analysis
- Profit Analysis
- Product Category Filter
- KPI Summary

---

### 📦 Dashboard 2 – Retail Operations Dashboard

**Focus Areas**

- Category-wise Analysis
- Sub-Category Performance
- Profitability Analysis
- Inventory Insights
- Top Products
- Business Performance Comparison

---

## 🎯 Objectives

- Clean and organize the retail dataset.
- Calculate Total Sales and Profit.
- Analyze monthly sales performance.
- Analyze region-wise sales.
- Evaluate category and sub-category performance.
- Build two interactive dashboards.
- Connect all visualizations using Combo Box filters.
